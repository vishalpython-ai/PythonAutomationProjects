# 🚀 Hacker News Scraper

A Python script that scrapes [Hacker News](https://news.ycombinator.com/) to extract **popular news articles** based on points.  
This project demonstrates **web scraping, data parsing, and Python automation** — ideal for freelancers and developers showcasing their skills.

---

## 📌 Features
- Scrapes **latest Hacker News articles**.
- Extracts:
  - News Title
  - Hyperlink (URL)
  - Points (upvotes)
- Filters articles by **minimum points** (default: 100).
- Clean and professional Python implementation.
