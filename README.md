# 📧 Email Scraper

A simple Python automation project that extracts **email addresses** and **phone numbers** from text files, pdf files, word files, using **Regular Expressions (Regex)**.  
This project is part of my Python & Automation learning journey 🚀.

---

## ⚡ Features
- Extracts **valid email addresses** 📧
- Extracts **phone numbers** (Indian & International formats) 📱
- Works with plain `.txt` files
- Easy to extend for CSV/Excel integration

---

## 🛠️ Tech Stack
- **Python 3.10+**
- **Regex (re module)**
- **File handling**

---

## 📂 Project Structure
EmailScraper/
│-- input_files/ # Folder to keep input text files
│-- output_files/ # Folder to store extracted results
│-- email_scraper.py # Main script
